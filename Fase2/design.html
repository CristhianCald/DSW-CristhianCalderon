Propósito del sitio

Con el desarrollo de este objetivo virtual se busca generar conocimiento en el área de programación web con el uso de etiquetas básicas de HTML, atributos y valores que constituyen conceptos bases para el desarrollo de sitios web y generación de contenido de los mismos. Se abordarán temas de gran importancia que junto a la práctica perimtiran fortalecer los conocimientos adquiridos en la fase inicial de este objetivo virtual.

Etiquetas HMTL

Las etiquetas HTML comprenden una fragmento de código que permite la creación de elementos y estructuración de los contenidos expuestos en este lenguaje, comprendiendo la especificación de las etiquetas permitiendo su designación y función específica teniendo en cuenta el tipo de información que queremos mostrar. Existen etiquetas semánticas, de contenido multimedia, formularios que en conjuntos abarcan los contenidos de información y estructuración de un sitio web desarrollado en el lenguaje. Celaya Luna, A. (2014)

Atributos y valores

Los atributos y valores se encuentran directamente relacionados con las etiquetas ya que explicado de una forma más sencilla, un atributo se puede decir que establece una propiedad de una etiqueta y el valor define la propiedad como tal, es así como los atributos y valores componen la practicidad de una etiqueta y el uso que se le puede dar a los mismos en el desarrollo de sitios web.
